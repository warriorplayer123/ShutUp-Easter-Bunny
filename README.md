# 🐰 ShutUp Easter Bunny

Small TERA module that hides annoying messages from event bunny NPCs.

---

## 🚀 Usage

### Toggle on/off:

```id="yznr2g"
shush
```

* ON → hides bunny messages (enabled by default)
* OFF → shows them again

---
## ⚠️ Notes

* Only affects specific bunny NPCs
* You can edit `config.json` to change IDs \ or u can modify this mod to mute all NPCs
* Safe to toggle anytime
